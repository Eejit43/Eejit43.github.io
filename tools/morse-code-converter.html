Morse Code Converter:

Morse Code can be represented in writing using dashes and dots (hyphens/underscores and periods, -/_ and .). Characters are separated by spaces, and actual spaces are represented by a backslash (/), a vertical slash (|), or three spaces ( ). For example, test abc converts to - . ... - .- -... -.-.. Morse code does not keep character case. This has been expanded to include conversions not officially recognized by ITU-R M.1172 or ITU-R M.1677-1 (;, _, and $).
